#### Curora Medical Surgery - Backend

#### **Project run:**
- 1. To run local server, first install node_modules
```
yarn install
```
- 2. Now run this command
```
yarn start:dev
```
- 3. To run production server, run this two commands
```
yarn build
yarn start:prod
```
- 4. Code formatting
```
yarn prettier:fix
```
- 5. Check errors
```
yarn lint:fix
```
